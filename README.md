# Bright-TV-analysis
Case Study, clean dataset
BrightTV is focused on expanding its subscription base during the current financial year. To achieve this, the Customer Value Management (CVM) team requires actionable insights into user behavior and content consumption patterns. Analysis of subscriber profiles and detailed viewership sessions reveals clear trends in engagement, peak usage times, and demographic preferences.

Key findings highlight that consumption is strongly influenced by time of day, content type, and device usage. Low‑consumption periods can be addressed through targeted programming such as family content in the evenings, short‑form entertainment midweek, and localized offerings during holidays. Growth opportunities lie in personalized recommendations, referral incentives, and partnerships to broaden reach.

This project delivers a 20‑minute presentation supported by SQL‑driven analysis, dashboards, and strategic recommendations. The outcome is a data‑driven roadmap enabling BrightTV to increase engagement, optimize content strategies, and accelerate subscriber growth
